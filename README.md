# datasmart-utils

> Teste de componentes

## Instalação

```bash
npm install --save datasmart-utils
```

ou

```bash
yarn add datasmart-utils
```

## Usage

```jsx
import React from "react";

import { Teste1, Teste2, Teste3 } from "datasmart-utils";

function App{
	return (
		<>
			<Teste1 />
			<Teste2 />
			<Teste3 />
		</>
	);
}
```
